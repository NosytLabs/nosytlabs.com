## 🔧 **Troubleshooting**

### **Build Issues**
```bash
# If build fails, check for syntax errors
npm run build

# Fix CSS issues
npm run fix-css

# Check for TypeScript errors
npm run type-check
```

### **Development Server Issues**
```bash
# If port 3000 is in use, server will use 3001
# Check terminal output for actual port

# Restart server if needed
Ctrl+C (stop)
npm run dev (restart)
```

### **Performance Issues**
```bash
# Run performance analysis
node scripts/live-site-testing.js

# Check bundle size
npm run build
# Look for bundle size in output
```
