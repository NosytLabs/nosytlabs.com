## 1. Understanding Tokens

### What's a Token?
A token is a piece of text that ChatGPT reads or writes. Roughly:
- 1 token ≈ 4 characters
- 1 token ≈ 0.75 words
- 100 tokens ≈ 75 words

### Real Token Examples

**"Hello, how are you?"**
→ ~5 tokens
→ Cost: $0.00001 (GPT-4o mini input)

**Customer support email response** (~150 words)
→ ~200 tokens
→ Cost: $0.00003 (input) + $0.0012 (output) = $0.00123

**Article summarization** (~1,000 words → 100-word summary)
→ 1,300 tokens input + 140 tokens output
→ Cost: $0.0011 input + $0.00084 output = $0.00194

## 5. Cost Optimization Tips

### 1. Choose the Right Model
- **GPT-4o Mini:** 100x cheaper, good for most tasks
- **GPT-4o:** 30x cheaper than GPT-4, usually sufficient
- Only use expensive models when necessary

**Potential savings:** 50-70% by right-sizing model choice

---

### 2. Batch Processing
Instead of real-time API calls, batch process during off-peak hours for 50% discount.

Example: Process 1,000 documents overnight at 50% discount instead of real-time.

---

### 3. Prompt Caching
Reuse expensive system prompts with cache (cache hits cost 10% of normal).

Example: Save 500 tokens × 100 chats/day = savings of $0.03/day = $9/month

---

### 4. Implement Filtering
- Reject bad inputs before API call
- Cache common responses
- Rate limit to prevent abuse

**Potential savings:** 20-30%

---

### 5. Use Fallback Logic
- Local rules for simple cases
- Only use API for complex tasks

Example: 80% of support questions are FAQ → use lookup table, only 20% need ChatGPT

**Potential savings:** 40-60%

## 8. FAQ: ChatGPT Pricing

### How is billing calculated?
By tokens: Every token (input and output) is counted and charged. You pay per token used, divided by 1,000.

### Can I set a spending limit?
Yes, set hard limits in OpenAI dashboard. Requests will fail if exceeded.

### What about rate limiting?
Start with 3,500 requests/minute (GPT-4o). Higher limits available on request.

### Is my data private?
Your API calls are not used for training (OpenAI's privacy policy). But data still goes to OpenAI's servers. For sensitive data, consider alternative models.

### Can I use ChatGPT offline?
No, ChatGPT API requires internet connection. You could use local models (Llama, Mistral) for offline use.

### How do I estimate costs before launch?
Request a test API key, run a sample of your data through the API, and extrapolate costs based on token usage.
